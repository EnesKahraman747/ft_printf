# ft_printf Projesi

42 Okulu'nda öğrencilere verilen ft_printf projesi, özel bir `printf` işlevini yeniden uygulamayı amaçlar. 
Bu proje, C dilinde biçimlendirilmiş çıktı üretme yeteneğinizi geliştirmenize olanak tanır ve veri türlerini 
işleyerek metni biçimlendirmenin temellerini öğretir.

## Proje Hakkında

ft_printf projesi, aşağıdaki temel bölümleri içerir:

- **Biçim Dizgesi İşleme**: `printf` benzeri biçim dizgelerini analiz etme ve işleme yeteneği.
- **Veri Türlerini İşleme**: `%d`, `%s`, `%c`, `%f`, vb. gibi veri türlerini işleyebilme yeteneği.
- **Genişletilebilirlik**: Özel biçim belirleyicileri ekleyebilme yeteneği (örneğin, `%b` için ikili sayılar).
- **Kapsamlı Hata İşleme**: Hatalı biçim dizgelerini ve işlemlerini yakalama yeteneği.

Bu proje, C programlama dilini daha iyi anlamanıza yardımcı olurken, biçimlendirilmiş çıktı işleme yeteneğinizi geliştirir ve işe alım süreçlerinde sizi öne çıkarabilir.
